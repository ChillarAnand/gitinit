gitinit
=======

Initiates an empty git repository with a `.gitignore` file for the provided language.

Installation
------------

    $ [sudo] pip install gitinit

Usage
-----
    
    # initiates with a generic .gitignore file
    $ gitinit

    # initiates a Haskell .gitignore file
    $ gitignore -l haskell

Help
----

    $ gitinit -h
